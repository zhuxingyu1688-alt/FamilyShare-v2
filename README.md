# 家庭共享 - 双定时深度优化版 v2.0

这是一个透明、可关闭的家庭位置共享 Android App，支持每天最多两个定时发送位置。

## v2.0 新增

-   **双定时**：支持两个独立的每日发送时间（定时① + 定时②）
-   **独立闹钟**：两个闹钟互不影响，触发时只重排自己
-   **网络恢复**：断网后重连自动恢复两个闹钟
-   **发送重试**：网络失败自动重试 2 次（间隔 3 秒）
-   **OkHttp 单例**：减少重复创建，降低内存开销
-   **双状态显示**：首页同时显示两个定时的下次计划

## 核心保证

- App 名称「家庭共享」
- 只在到点时间窗口或手动测试时访问定位
- 开机恢复、保存时间、切后台不会访问定位
- 高精度 GPS 优先，最多等 2 分钟，精度差不乱发
- 手机定位开关关闭时，只推提醒不偷开
- 发送/失败后自动停止前台定位，省电

## 飞书机器人建议

- 关键词：位置
- 签名校验：先关闭
- IP 白名单：先关闭

## 荣耀手机必须设置

设置 → 应用 → 应用启动管理 → 家庭共享

- 自动管理：关闭
- 允许自启动：开启
- 允许关联启动：开启
- 允许后台活动：开启

还要检查：

- 位置权限：始终允许 + 精确位置
- 通知权限：允许
- 闹钟和提醒/精确提醒：允许
- 电池优化：不允许优化/无限制

## 重要限制

如果用户在系统设置里点「强行停止」App，Android 不允许 App 自己复活。必须手动打开一次 App。

如果手机定位总开关被手动关闭，普通 App 不能后台偷偷打开定位开关，只能提醒用户手动开启。

## GitHub Actions 打包

上传整个项目到 GitHub 后，进入 Actions，运行 `Build Android APK Final Pro`。

成功后优先找：

- Code 页面里的 `apk-output/FamilyShare-final-all-in-one.apk`

如果没有，就进入 Actions 成功任务底部的 Artifacts，下载：

- `FamilyShare-final-all-in-one-apk`

解压后得到 `app-debug.apk`。
